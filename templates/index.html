Supported file types: HTML, CSS, JavaScript, Python, React, MongoDB-related scripts and queries.

How it works

  1. Upload ZIP
  2. Extract to local workspace
  3. Detect language by extension and content
  4. Run the matching analyzer stack
  5. View results and export PDF

Analyzer model

  • Each language has its own analyzer module
  • Each analyzer is split into rule files
  • Scores are calculated from accumulated rule penalties
  • Results are merged into one table
